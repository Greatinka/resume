A/B-тестирование эффективности рекламной кампании

Учебный pet-проект, направленный на освоение методологии A/B-тестирования и статистического анализа данных. 
В рамках проекта выполнено сравнение контрольной и тестовой групп пользователей с применением классических 
статистических методов для проверки гипотез.

В ходе реализации проекта были выполнены следующие этапы:
- генерация синтетических данных для двух групп (контрольной и экспериментальной);
- проведение описательного анализа и визуализация распределений;
- проверка статистических гипотез с использованием t-теста, U-теста Манна-Уитни и z-теста для пропорций;
- вычисление размера эффекта (Cohen's d).

Сгенерированные данные носят случайный характер, поэтому полученные результаты не обладают практической 
значимостью. Проект создан с целью демонстрации подходов к анализу данных и проведению A/B-экспериментов.

Использованные технологии: Python, pandas, NumPy, SciPy, Matplotlib, seaborn.
