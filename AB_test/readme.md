В ходе проекта были смоделированы данные для двух групп пользователей (контрольной и тестовой), рассчитаны описательные статистики
и проведены проверки гипотез с помощью t-теста, U-теста Манна–Уитни и z-теста для бинарной метрики.

Полученные результаты показывают небольшие статистические различия между группами, однако данные были сгенерированы случайным образом, 
поэтому они не отражают реального поведения пользователей. Цель проекта — продемонстрировать понимание принципов A/B-тестирования, 
работы с данными и применения статистических методов на практике.
