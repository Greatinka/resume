# A/B-тестирование эффективности рекламной кампании

## Цель проекта

Освоение методологии A/B-тестирования и статистического анализа данных в рамках учебного pet-проекта. Реализовано сравнение контрольной и тестовой групп пользователей с применением классических статистических методов для проверки гипотез.

## Этапы реализации

В ходе выполнения проекта реализован следующий комплекс задач:

- **Генерация данных:** создание синтетических наборов данных для контрольной и экспериментальной групп
- **Разведочный анализ:** проведение описательного анализа и визуализация распределений признаков
- **Статистическая проверка гипотез:** применение t-теста, U-теста Манна-Уитни и z-теста для пропорций
- **Оценка эффекта:** вычисление размера эффекта (Cohen's d)

## Характер проекта

Сгенерированные данные носят случайный характер, вследствие чего полученные результаты не обладают практической значимостью. Проект разработан с демонстрационной целью — для наглядной иллюстрации подходов к анализу данных и методологии проведения A/B-экспериментов.

## Технологический стек

- **Язык программирования:** Python
- **Обработка данных:** pandas, NumPy
- **Статистический анализ:** SciPy
- **Визуализация:** Matplotlib, seaborn
